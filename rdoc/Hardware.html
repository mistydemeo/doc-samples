cores_as_words() click to toggle source

Returns a string representing the number of CPU cores in English, suitable for use in user-readable output.

# File hardware.rb, line 56
def self.cores_as_words
  case Hardware.processor_count
  when 1 then 'single'
  when 2 then 'dual'
  when 4 then 'quad'
  else
    Hardware.processor_count
  end
end

MacCPUs

family() click to toggle source

Returns a symbol representing the family of CPU being used. Values are symbols, suitable for use with the optimization_flags hash. If the CPU family isnâ€™t recognized but is of a recognized CPU type, returns :dunno. Otherwise returns nil.

# File mac/hardware.rb, line 44
def family
  if type == :intel
    @intel_family ||= `/usr/sbin/sysctl -n hw.cpufamily`.to_i
    case @intel_family
    when 0x73d67300 # Yonah: Core Solo/Duo
      :core
    when 0x426f69ef # Merom: Core 2 Duo
      :core2
    when 0x78ea4fbc # Penryn
      :penryn
    when 0x6b5a4cd2 # Nehalem
      :nehalem
    when 0x573B5EEC # Arrandale
      :arrandale
    when 0x5490B78C # Sandy Bridge
      :sandybridge
    when 0x1F65E835 # Ivy Bridge
      :ivybridge
    else
      :dunno
    end
  elsif type == :ppc
    @ppc_family ||= `/usr/sbin/sysctl -n hw.cpusubtype`.to_i
    case @ppc_family
    when 9
      :g3  # PowerPC 750
    when 10
      :g4  # PowerPC 7400
    when 11
      :g4e # PowerPC 7450
    when 100
      :g5  # PowerPC 970
    else
      :dunno
    end
  end
end

Returns a boolean representing the value obtained from the sysctl kernel state tool.

# File mac/hardware.rb, line 123
def sysctl_bool(property)
  result = nil
  IO.popen("/usr/sbin/sysctl -n #{property} 2>/dev/null") do |f|
    result = f.gets.to_i # should be 0 or 1
  end
  $?.success? && result == 1 # sysctl call succeded and printed 1
end
